# RefMet

The main objective of RefMet (A Reference list of Metabolite names) is to provide a standardized reference nomenclature for both discrete metabolite structures and metabolite species identified by spectroscopic techniques in metabolomics experiments. RefMet can be seen as a mapping system, where, for a given metabolite name, we provide the associated IDs from other databases such as PubChem, KEGG and INCHI Key. More details can be found at https://www.metabolomicsworkbench.org/databases/refmet/index.php. 
The RefMet data is contained in the excel file in this folder.
